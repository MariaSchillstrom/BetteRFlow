# 2. Produkten

### 2. Produkten

#### 2.1 Översikt

BetteRFlow är en fullstack-webbapplikation byggd med ASP.NET Core Web API (backend) och Blazor Web App (frontend). Systemet hanterar två huvudsakliga användartyper:

1. **BRF-styrelsemedlemmar**: Registrerar fastigheter och fyller i omfattande formulär
2. **Mäklare**: Söker efter fastigheter och köper tillgång till data (planerat fas 2)

#### 2.2 Problem som löses

**Nuvarande situation:**

* Mäklare skickar manuellt PDF-formulär till BRF:er via mejl
* BRF:er fyller i PDF-dokument manuellt
* Informationen är ostrukturerad och svår att söka i
* Tidskrävande process för båda parter
* Risk för felaktig eller ofullständig information

**BetteRFlows lösning:**

* **Digital formulärhantering** med validering i realtid
* **Strukturerad datalagring** i relationsdatabas
* **Sökbar databas** för mäklare (planerat)
* **Automatiserad process** från registrering till åtkomst
* **Kvalitetssäkring** genom validering och tvingande fält

#### 2.3 Arkitekturval

Projektet använder **Clean Architecture** med tydlig separation of concerns:

**Motivering för arkitekturvalet:**

* **Tydlig separation**: Presentation, logik och data är separerade
* **Återanvändbar kod**: Shared-projektet innehåller modeller som används av både frontend och backend
* **Testbarhet**: Enkelt att testa varje lager isolerat
* **Skalbarhet**: Lätt att lägga till nya funktioner utan att påverka befintlig kod
* **Underhållbarhet**: Koden är organiserad och lätt att förstå
