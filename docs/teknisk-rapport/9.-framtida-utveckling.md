# 9. Framtida utveckling

Projektet är uppdelat i två faser. Efter delredovisningen den 19 december fortsätter utvecklingen med fas 2.

#### 9.1 Fas 2 (efter 19/12)

**Sökfunktion för mäklare**

* Sök efter fastigheter baserat på adress, område eller BRF
* Snabb överblick över tillgängliga fastigheter
* Filtreringsfunktioner

**Användarprofilhantering**

* Möjlighet för användare att uppdatera sin profil
* PUT-endpoint för användaruppdatering
* "Redigera profil"-sida i frontenden
* Utökad RegisterDto med Id-fält för UPDATE-operationer

**Azure Deployment**

* Deploy till Azure App Service
* Azure SQL Database i produktion
* CI/CD via GitHub Actions
* Azure Bicep för Infrastructure as Code

#### 9.2 Långsiktig vision

* **Notifikationssystem**: Email/SMS när formulär är ifyllda eller köp genomförda
* **Dokumentgenerering**: Automatisk generering av PDF-rapporter från formulärdata
* **API för externa system**: Möjlighet för mäklarsystem att integrera via API
* **Mobilapp**: React Native-app för mobil åtkomst
* **Dynamiska formulär**: Admin kan skapa nya formulär via gränssnittet

**JWT-autentisering**

* Implementera JSON Web Tokens för säkrare autentisering
* Tokens med expiration
* Refresh tokens
* Protected routes på backend

**Betalningsintegration**

* Stripe eller Swish för betalningar
* Mäklare köper tillgång till fastighetsdata
* Betalningshistorik
