# 3. Verktyg & teknologier

### 3. Verktyg och Teknologier

#### 3.1 Backend

**.NET 8**

* Senaste stabila versionen av .NET
* Förbättrad prestanda jämfört med tidigare versioner
* Long-term support (LTS)

**ASP.NET Core Web API**

* RESTful API-utveckling
* Inbyggt stöd för dependency injection
* Cross-platform (Windows, Mac, Linux)

**Entity Framework Core 9**

* Modern ORM (Object-Relational Mapping)
* Code-first approach
* Automatisk generering av databasmigrations

**SQLite (utveckling)**

* Lätt databas för lokal utveckling
* Inget serverinstallation krävs
* Enkel att versionshantera

**Azure SQL (produktion - planerat)**

* Skalbar molndatabas
* Automatisk backup
* High availability

#### 3.2 Frontend

**Blazor Web App (.NET 8)**

* C#-baserad frontend - samma språk som backend
* Komponentbaserad arkitektur
* Server-side rendering för bättre prestanda

**MudBlazor**

* Modernt UI-komponentbibliotek
* Material Design
* Välgjorda komponenter utan att designa från scratch
* Omfattande dokumentation

**Bootstrap**

* Responsiv layout
* Grid system
* Välkänt och väldokumenterat

#### 3.3 Verktyg

**Visual Studio 2022**

* Primär IDE
* Utmärkt .NET-support
* Integrerad debugging

**Git & GitHub**

* Versionshantering
* Samarbetsverktyg
* Projektdokumentation

**Swagger/OpenAPI**

* Automatisk API-dokumentation
* Interaktiv API-testning
* Generering av klientkod

**DB Browser for SQLite**

* Visuell databashantering
* Verifiering av databasstruktur
* Användes för att kontrollera migrations

**Draw.io**

* Skapande av arkitekturdiagram
* ER-diagram
* Flödesscheman

**GitBook**

* Teknisk dokumentation
* Versionshantering av dokumentation
* Sökbar kunskapsbas

#### 3.4 Varför dessa val?

**.NET-ekosystemet**

* Enhetligt språk (C#) över hela stacken
* Stark typning minskar buggar
* Omfattande standardbibliotek

**Blazor**

* Ingen JavaScript-komplexitet
* Snabbare utveckling med C#
* Komponentbaserad återanvändning

**Entity Framework**

* Minskad boilerplate-kod
* Automatiska migrations
* Enkel databashantering

**MudBlazor**

* Professionellt utseende utan designarbete
* Färdiga komponenter (formulär, tabeller, dialoger)
* Aktivt community och support
