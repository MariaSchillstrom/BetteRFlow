# 1. Inledning

### **Syfte:**&#x20;



BetterBFlow är ett system för att samla in och sälja fastighetsdata från bostadsrättsföreningar till mäklare.

### **Mål:**&#x20;



Automatisera insamling av BRF-data och skapa en enkel köpplattform för mäklare.

### **Funktion:**&#x20;



BRF fyller i formulär → Data sparas → Mäklare kan köpa access med volymrabatt.
