# 4. Roller & fördelar med verktygen

### Roller&#x20;

#### .NET & C\#

**Roll:** Grundplattform och språk&#x20;

**Vad de gör:** Möjliggör att kod skrivs en gång och körs överallt

#### [ASP.NET](http://asp.net) Core Web API

**Roll:** Backend/Server&#x20;

**Vad det gör:** Hanterar affärslogik, säkerhet och databaskommunikation

#### Blazor

**Roll:** Frontend/Användargränssnitt

**Vad det gör:** Skapar det användaren ser och interagerar med

#### Entity Framework Core

**Roll:** Databasabstraktion **V**

**ad det gör:** Översätter mellan kod och databas

#### MudBlazor

**Roll:** UI-komponentbibliotek&#x20;

**Vad det gör:** Ger färdiga, snygga designkomponenter

### Fördelar&#x20;

#### För utveckling:

✅ **Snabbare utveckling**

* Färdiga komponenter (MudBlazor)
* Automatisk databaskod (Entity Framework)
* Ett språk för allt (C#)

✅ **Färre buggar**

* Stark typning i C#
* Automatisk validering
* Vältestad kod från Microsoft

✅ **Enklare underhåll**

* Tydlig struktur
* Versionskontroll (Git)
* Automatiska tester möjliga

#### För verksamheten:

✅ **Kostnadseffektivt**

* Gratis verktyg (open source)
* Snabbare utveckling = lägre kostnad
* Lättare att hitta utvecklare som kan C#/.NET

✅ **Säkerhet**

* Microsofts säkerhetsuppdateringar
* Inbyggt skydd mot vanliga attacker
* Kryptering av lösenord

✅ **Skalbarhet**

* Kan hantera många användare samtidigt
* Lätt att flytta till molnet (Azure)
* Prestanda optimerad

✅ **Framtidssäkert**

* Long-term support från Microsoft
* Stort community
* Regelbundna uppdateringar
