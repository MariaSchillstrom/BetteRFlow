# 2. Översikt av verktyg

### Översikt av verktyg

BetteRFlow använder följande huvudverktyg:

**Programmeringsspråk:**

* C# (C-Sharp) - Microsofts moderna programmeringsspråk

**Backend (servern):**

* .NET 8 - Plattform för att köra applikationer
* [ASP.NET](http://asp.net) Core Web API - Ramverk för att skapa API:er

**Frontend (användargränssnittet):**

* Blazor - Ramverk för att bygga interaktiva webbsidor med C#
* MudBlazor - Färdiga designkomponenter

**Databas:**

* Entity Framework Core - Verktyg för att prata med databaser
* SQLite (utveckling) / Azure SQL (produktion) - Databashantering
