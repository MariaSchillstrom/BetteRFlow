# 1. Syfte & målgrupp

### Syfte

När man bygger webbapplikationer behöver utvecklare olika verktyg och ramverk för att effektivisera arbetet. Istället för att skriva all kod från scratch använder man färdiga verktyg som löser vanliga problem och ger struktur åt projektet.

BetteRFlow använder moderna Microsoft-verktyg som är särskilt lämpade för att bygga säkra och skalbara affärssystem.

### Målgrupp

Denna presentation riktar sig till:

* **Projektledare och beslutsfattare** som vill förstå teknikvalen
* **Intressenter** som behöver översikt över projektets tekniska grund
* **Icke-teknisk personal** som ska förstå systemets uppbyggnad
* **Studenter** som lär sig om webbutveckling
