# BetteRFlow

<figure><img src=".gitbook/assets/brf (1).jpg" alt=""><figcaption></figcaption></figure>

<h3 align="center"></h3>

<p align="center">Denna rapport är en del av examensarbetet för utbildningen .NET-Utvecklare vid Campus Mölndal. Rapporten beskriver utvecklingen av BetteRFlow, en webbapplikation för att effektivisera informationsflödet mellan bostadsrättsföreningar och mäklare.</p>

<p align="center">Rapporten fokuserar på den tekniska implementationen, samt verktyg fram till första delredovisningen den 19 december 2025. För djupare teknisk dokumentation av specifika klasser och metoder, se kommentarer i kodbasen samt projektets GitHub-repository.</p>
